# Pathfinder2026 Workshop

Start with Phase 1 and work down the list.

| Phase | Start Here | Goal |
|-------|------------|------|
| Phase 1 | [Assemble](ASSEMBLE.md) | Set up the Pi 500, assemble the robot, connect, and test hardware |
| Phase 2 | [Capabilities Exploration](CAPABILITIES_EXPLORATION.md) | Learn the robot code by running and editing capability demos |
| Phase 3 | [Course Challenge](COURSE_CHALLENGE.md) | Use the robot capabilities on the course |

For the workshop, the Pi 500 and robot images should already be created. Skip image-build documents during the event.

If something fails, use [Student troubleshooting](../support/TROUBLESHOOTING.md). Compare with another team first. Get help right away if the issue involves wiring, batteries, heat, smoke, broken parts, or `sudo` commands.
