## Field Direction

Areas are numbered counter-clockwise:

1. Area 1: bottom right, tag 582, start square and blue blocks
2. Area 2: top right, tag 583, red blocks
3. Area 3: top left, tag 584, yellow blocks
4. Area 4: bottom left, tag 585, delivery zone

## Before Finalizing

- Build a small mockup before event day.
- Confirm each AprilTag is detected by the robot camera.
- Confirm the start square and delivery zone are obvious from standing height.
- Test lighting with the actual blue, red, and yellow blocks.
- Keep scoring simple until the physical course is proven.
